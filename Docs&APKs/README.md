# LipiSathi — Internet-Free Indian Transliteration

Built by Team Word Weavers for Smart India Hackathon 2025 (SIH25155).

LipiSathi helps users read Indian signboards in their familiar script by transliterating text across Indian languages while preserving pronunciation — completely offline.

## Features
- Offline transliteration
- Automatic script detection
- Multi-script support
- Hub-and-spoke transliteration architecture
- Exception dictionary handling
- Indic language support:
  - Telugu
  - Hindi / Devanagari
  - Tamil
  - Kannada
  - Malayalam
  - Bengali
  - Gujarati
  - Odia
  - Gurmukhi
